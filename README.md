# Дипломный проект по курсу «Автоматизатор тестирования»

## Описание проекта
Приложение — это комплексный сервис «Путешествие дня», который предлагает купить тур по определенной цене с помощью банковской карты. Покупка возможна двумя способами:
1. Обычная оплата по дебетовой карте.
2. Оформление кредита по данным банковской карты.

Сервис взаимодействует с банковскими сервисами (Payment Gate и Credit Gate) и сохраняет информацию о платежах в собственной базе данных.

## Технические требования
Для запуска проекта и тестов на вашем компьютере должны быть установлены:
* **Java**: Версия 21 (рекомендуется GraalVM или OpenJDK).
* **Docker Desktop**: Обязателен для запуска контейнеров с базами данных (MySQL/PostgreSQL).
* **Браузер**: Google Chrome (для работы Selenide).

---

## Инструкция по запуску

### 1. Подготовка окружения (Базы данных)
Перед запуском тестов необходимо поднять контейнеры с базами данных. В корне проекта выполните команду:
```bash
docker-compose up -d

```

### 2. Запуск приложения (SUT)

Для запуска самого веб-сервиса выполните команду (приложение будет доступно по адресу `http://localhost:8080`):

```bash
java -jar aqa-shop.jar

```

*Примечание: Убедитесь, что переменная окружения JAVA_HOME настроена на JDK 21.*

### 3. Запуск автотестов

Чтобы запустить автоматизированные тесты, откройте новый терминал и выполните:

```bash
./gradlew clean test

```

*Если тесты падают из-за обнаруженных багов в приложении — это ожидаемое поведение.*

### 4. Работа с отчётами Allure

Для генерации и просмотра интерактивного отчёта в браузере используйте команду:

```bash
./gradlew allureServe

```

## Реализованные проверки

В проекте реализованы UI-тесты с использованием **Selenide** для следующих сценариев:

* Покупка тура по карте со статусом `APPROVED`.
* Отказ в покупке тура по карте со статусом `DECLINED`.
* Проверки валидации полей формы (номер карты, срок действия, владелец, CVC).

## Используемые инструменты

* **Java 21** — основной язык разработки.
* **Gradle** — система сборки и управления зависимостями.
* **JUnit 5** — тестовый фреймворк.
* **Selenide** — фреймворк для лаконичных UI-тестов.

* **Docker & Docker Compose** — запуск инфраструктуры (БД).
* **Lombok** — для сокращения шаблонного Java-кода.
