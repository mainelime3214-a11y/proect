# Отчёт по итогам автоматизации

## Что было запланировано и реализовано
1. **Планирование**: Составлен план автоматизации (Plan.md).
2. **Инфраструктура**: Настроено окружение (Java 21, Gradle, Docker).
3. **Автоматизация**:
    * Реализованы Page Objects (MainPage.java) для взаимодействия с UI.
    * Написаны UI-тесты с использованием Selenide.
    * Интегрированы отчеты Allure.
4. **Отчетность**: Подготовлены итоговые документы по результатам тестирования.

## Причины расхождений
Основные задержки были вызваны настройкой JDK 21 и конфигурацией Gradle (ошибки путей и кодировки), что потребовало дополнительного времени на отладку среды.

## Сработавшие риски
* Сложности с установкой специфической версии Java (GraalVM).
* Ошибки кодировки UTF-8 в именах селекторов.

## Общий итог по времени
* **Планировалось:** 16 часов.
* **Потрачено:** 20 часов (из-за настройки окружения и Docker).
